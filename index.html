Features

Game World Design

The game has floor, roof, wall texture and tile map designs for the cyborg to have an environment to play within. 

Collision Detection

The game has the ability to sense whether a cyborg has hit a collision (wall) or has been attacked by an enemy with a weapon.

Ray Casting & Player Control

A user has full control of the players moves. It determines how wide the player sees the world in front of him/her.

About

The Maze is a 3D Maze game that uses ray casting to render a 2D map into a 3D navigable world!

The Maze was written was written in C ussing SDL2 library. Deveploment was performed using Ubuntu 14.04 LTS - gcc (Ubuntu 4.8.4-2ubuntu1~14.04) 4.8.4Simple mathematical algorithms and basic physics was applied in game design. Pythagoras theorem and rule of component decomposition was applied in the game to design the wall in the Game world to change the 2D-form of the game to 3D-Game using the concept of ray casting.
